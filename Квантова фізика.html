Конструктор промптів за технікою PRECISION

Завдання: Пояснення квантової фізики

Для учнів 10 класу

Дисципліна: Інформатика / Фізика

Опис завдання:

Тема: Пояснення складної теми

Мета: Скласти промпт, щоб AI пояснив квантову фізику простою мовою для презентації в класі

Очікуваний результат: Зрозуміле пояснення з аналогіями та візуальними прикладами

Інструкція для виконання:

  1. Оберіть по одному варіанту з кожного розділу таблиці
  2. Запишіть обрані варіанти в окремий документ
  3. Сформулюйте промпт, об'єднавши всі компоненти
  4. Протестуйте промпт з AI (ChatGPT, Claude, Gemini)
  5. Проаналізуйте якість отриманої відповіді
Літера Компонент Варіанти для вибору
P Purpose
(Призначення/Мета)
Вариант 1: Шкільна презентація
Готую презентацію про квантову фізику для виступу перед однокласниками на уроці фізики
Вариант 2: Науковий проект
Створюю матеріали для шкільного наукового проекту з квантової механіки
Вариант 3: Популяризація науки
Підготовка матеріалу для популяризації квантової фізики серед підлітків
Вариант 4: Дослідницька робота
Розробка розділу про квантову фізику для дослідницької роботи МАН
Вариант 5: Інтерактивний урок
Створення інтерактивного уроку з елементами квантової фізики для молодших класів
Вариант 6: Конкурсна робота
Підготовка до участі в конкурсі наукових проектів з фізики
R Role
(Роль учасників)
Вариант 1: Експерт і учень
Ти провідний фізик-теоретик, я учень 10 класу, який вперше вивчає квантову фізику
Вариант 2: Популяризатор науки
Ти науковий журналіст, який спеціалізується на поясненні складних наукових концепцій
Вариант 3: Досвідчений вчитель
Ти вчитель фізики з 20-річним досвідом пояснення квантової механіки старшокласникам
Вариант 4: Наставник-дослідник
Ти молодий науковець, який може зрозуміло пояснити сучасні досягнення квантової фізики
Вариант 5: Інноваційний педагог
Ти педагог-новатор, який використовує креативні методи навчання фізики
Вариант 6: Ментор STEM
Ти ментор STEM-освіти, який надихає підлітків на вивчення точних наук
E Expectations
(Очікування якості)
Вариант 1: Максимальна зрозумілість
Очікую найпростіших пояснень без складної математики та термінології
Вариант 2: Наукова точність
Потрібна наукова достовірність з адаптацією для шкільного рівня
Вариант 3: Креативність подачі
Хочу оригінальні аналогії та незвичайні способи пояснення
Вариант 4: Інтерактивність
Потрібні елементи, які можна показати та продемонструвати в класі
Вариант 5: Мотивування інтересу
Важливо захопити увагу та показати захоплюючу сторону квантової фізики
Вариант 6: Практичне застосування
Хочу акцент на реальних застосуваннях квантової фізики у житті
C Constraints
(Обмеження)
Вариант 1: Без складної математики
Уникай рівнянь Шредінгера та складних математичних формул
Вариант 2: Доступна мова
Не використовуй професійну наукову термінологію без пояснень
Вариант 3: Вікова відповідність
Враховуй рівень знань та інтереси 15-16-річних підлітків
Вариант 4: Часові рамки
Матеріал має поміститися в 10-хвилинну презентацію
Вариант 5: Доступні ресурси
Використовуй лише ті демонстрації, які можна зробити в школі
Вариант 6: Позитивний настрій
Уникай складних філософських питань про природу реальності
I Information
(Ключова інформація)
Вариант 1: Основні принципи
Суперпозиція, квантова заплутаність, принцип невизначеності Гейзенберга
Вариант 2: Історичний контекст
Відкриття Планка, досліди Резерфорда, внесок Ейнштейна та Бора
Вариант 3: Сучасні застосування
Квантові комп'ютери, лазери, МРТ, квантова криптографія
Вариант 4: Дивні явища
Тунельний ефект, кіт Шредінгера, квантова телепортація
Вариант 5: Порівняння з класичною фізикою
Відмінності між квантовим та класичним світом, межі застосування
Вариант 6: Майбутні технології
Квантовий інтернет, квантові сенсори, перспективи розвитку
O Output
(Формат результату)
Вариант 1: Структурована презентація
План презентації з 7-10 слайдів, включаючи заголовки та ключові пункти
Вариант 2: Набір аналогій
Колекція з 5-7 простих аналогій для пояснення квантових концепцій
Вариант 3: Сценарій виступу
Детальний текст з підказками щодо жестів, пауз та акцентів
Вариант 4: Інтерактивні елементи
Опис експериментів та демонстрацій, які можна провести в класі
Вариант 5: Візуальний план
Опис схем, діаграм та ілюстрацій для наочного представлення
Вариант 6: Запитання для обговорення
Список запитань для залучення аудиторії та перевірки розуміння
N Notes
(Додаткові примітки)
Вариант 1: Цікаві факти
Включи дивовижні факти про квантовий світ та парадокси
Вариант 2: Зв'язок з повсякденням
Покажи, як квантова фізика впливає на звичайне життя підлітків
Вариант 3: Кар'єрні можливості
Розкажи про професії, пов'язані з квантовими технологіями
Вариант 4: Українські досягнення
Згадай внесок українських учених у розвиток квантової фізики
Вариант 5: Популярна культура
Використай приклади з фільмів, серіалів та ігор про квантову фізику
Вариант 6: Ресурси для поглиблення
Додай список відео, книг та сайтів для подальшого вивчення теми

Приклад сформованого промпту:

"Готую презентацію про квантову фізику для виступу перед однокласниками на уроці фізики (P). Ти вчитель фізики з 20-річним досвідом пояснення квантової механіки старшокласникам (R). Очікую найпростіших пояснень без складної математики та термінології (E). Уникай рівнянь Шредінгера та складних математичних формул (C). Тема: суперпозиція, квантова заплутаність, принцип невизначеності Гейзенберга (I). План презентації з 7-10 слайдів, включаючи заголовки та ключові пункти (O). Включи дивовижні факти про квантовий світ та парадокси (N)."
